# Kutuphane Chat

Basit ve kullanıcı dostu bir sohbet uygulaması. Kullanıcı adı ve profil fotoğrafı ile hızlıca sohbete başlayın.

## Özellikler

- ✅ Kullanıcı adı ve profil fotoğrafı ile giriş
- ✅ Gerçek zamanlı sohbet arayüzü
- ✅ Tüm veriler cookie'lerde saklanır (veritabanı gerekmez)
- ✅ Modern ve temiz tasarım
- ✅ Responsive tasarım

## Kurulum

```bash
npm install
npm run dev
```

## Kullanım

1. Kullanıcı adınızı girin
2. (Opsiyonel) Profil fotoğrafı seçin
3. Sohbete başlayın!

Tüm mesajlar ve kullanıcı bilgileri tarayıcı cookie'lerinde saklanır.
